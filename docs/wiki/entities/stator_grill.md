# Stator Grill

**Source**: `src/cad/stator/fan_grill_eng.scad`

## Description
Parametric fan grill (stator) with hybrid 2D engineering annotations.

## Critical Parameters
- **sizeXY**: Derived from `fanSize`
- **sizeZ**: Derived from `thickness`
- **innerGrillDiameter**: Derived from `hubDiameter`

## Performance Callouts
- **P1**: Tip Clearance (Variable) - Monitored for aerodynamic performance.
- **P2**: Grill Impedance Pattern.
